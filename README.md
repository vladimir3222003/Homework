# Землепользование и классификация участков Земли
В этом исследовании мы рассматриваем проблему землепользования и классификации растительного покрова, используя спутниковые снимки Sentinel-2. Спутниковые снимки Sentinel-2 находятся в открытом и свободном доступе в рамках программы наблюдения Земли Copernicus. Мы представляем новый набор данных, основанный на спутниковых снимках Sentinel-2, охватывающих 13 спектральных диапазонов и состоящих из 10 классов, в общей сложности 27 000 изображений с маркировкой и географической привязкой. Мы предоставляем эталоны для этого нового набора данных с его спектральными диапазонами, используя современную глубокую сверточную нейронную сеть (CNNs). С помощью предложенного нового набора данных мы достигли хорошей общей точности классификации. Полученная в результате система классификации открывает возможности для ряда приложений наблюдения Земли. Мы демонстрируем, как эта система классификации может быть использована для определения изменений в землепользовании и растительном покрове и как она может помочь в улучшении географических карт.

# Запуск проекта
1. Переходим в Notebook, настраиваем окружения: инсталирование ключевых библиотек, описанных в requirements
2. Обязательно подключаем графический процессор GPU
3. Последовательно выполняем в описанных ячейках кода:
   3.1 загрузку EuroSat.zip(в случае реализации собственных примеров это нужно будет для определения классов фото)
   3.2 создание кастомного датасета
   3.3 нормализация изображений по каналам RGB
   3.4 загрузку датасета с делением по классам
4. При выполнении шагов описанных выше загружаем саму модель
